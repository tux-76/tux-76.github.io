Description

The console playground for Buford2 allows for JavaScript developers to use Buford's functions in a programming environment.

To use: open the console and use the namespace "bu2" to access the function tree. This page loads in the "no-module" version of Buford2, which also includes extra private functions under "bu2._e".

To get the specific adresse of a function, navigate to it on the normal website and scroll to the "Details" box. From there you will find the adress and specific syntax of that function.

Also included is the original Buford for legacy's sake. It uses the namespace "bu".
