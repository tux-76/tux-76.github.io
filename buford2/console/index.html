Description

The console playground for Buford2 allows for JavaScript developers to use Buford's functions in a programming environment.

To use: open the console and use the namespace "bu2" to access the function tree. To see specific adresses, go the a function on the normal website and scroll to the "Details" box. From there you will find the adress and specific syntax of the function.

Also included is the original Buford for legacy's sake. It uses the namespace "bu".
