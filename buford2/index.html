Buford2 Home

Description

Buford2 is a JavaScript library and website meant to solve complex problems, mainly algebraic equations. Some algebraic functionalities include solving an equation for a variable, such as "20-2a=a-5", or more complex, multivariabled problems. Also included are factoring tools, such as the GCF, LCM, factorization, and prime factorization. This website is a GUI to access all the functions provided by Buford2, but neither the website nor the library are yet complete and some functions may be unusable.
